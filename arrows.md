## Just a cheat sheet for arrow symbols

I am just lazy to remember arrow symbols, so here is a cheat sheet.

* Vertical and Horizontal Arrows
Up:
```
↑
```
Down:
```
↓
```
Left:
```
←
```
Right:
```
→
```

* Angled Arrows (for intersections or diagonal moves)
Up-Left:
```
↖
```

Up-Right:
```
↗
```

Down-Left:
```
↙
```

Down-Right:
```
↘
```
